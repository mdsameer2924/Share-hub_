### Share Hub steps to run this software

As i expect you've extracted this tarball file 
now let's begin 

### Step 1 
Create `~/bin` directory
```bash
mkdir -p ~/bin 
```
### Step 2 
Move Binary file **Name:-** `sharehub` to `~/bin` directory 

### Step 3
Add home-user path to use binary
```bash
echo 'export PATH="$HOME/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

### Step 4- Verify installation
```bash
sharehub # to open app itself
```
or 
```bash
which sharehub #output expected /home/usrname/bin
```
